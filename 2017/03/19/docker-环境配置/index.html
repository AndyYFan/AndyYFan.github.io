##Docker环境配置

Docker是一个轻量级的容器技术,和传统的虚拟机技术有所不同，Docker直接运行在操作系统之上，而不是硬件层。
虽然是操作系统层上的虚拟机实现，但是也实现了资源隔离，同时也因此具有比较好的性能，一般来说，容器的启动一般在1秒左右。

OSX下可以下载到最新的docker

docker具有统一管理的镜像仓库，支持将软件编译成一个image，在image中进行配置后发布，使用者可以通过从repository中下载镜像。

可以上传和下载仓库中的镜像。
