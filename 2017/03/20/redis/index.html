Redis 初学

发表于 2017-03-20

###什么是Redis？
Redis是一个key-value的数据库，和memcached类似，但是除了可以存储String类型之外，还可以存储lists，sets,和ordered sets,这些数据类型可以通过原子的push/pop来操作，还包括add/remove, union，intersection，求集合不同，等等操作都是原子操作。

###Jedis
Jedis数据库连接池：
JedisPool pool = new JedisPool(new JedisPoolConfig(), "localhost");
也可以声明成静态对象，这是线程安全的。
JedisPool 基于Commons Pool 2 所以可以通过和Commons Pool类似的方式配置。
在实际应用中：

Redis 提供主从复制的机制，一个已经声明的Jedis连接实例可以通过两种方式声明主从关系
一种是通过xml配置文件的方式，另外一种是直接通过jedis.slaveOf("host_name","port")声明。
从Redis2.6开始从数据库默认是只读的，写需求会报错。就算修改从的设置，写请求不会备份，而且会被重写覆盖。
当主机down掉后，可以把从机改成master，可以通过slave.slaveOfNoOne()修改slave成为master，如果有不止一台slave，其他slave要通过slaveOf修改主从关系。

###SDR(Spring Data Redis)
SDR提供了很高层的抽象，所以可以很方便的使用redis。
使用Redis的第一步是通过IoC容器连接redis数据库。一般是org.springframework.data.redis.connection这个包。RedisConnection提供了和Redis交互的句柄，也可以把Redis的异常信息转换成Spring的consistent DAO exception hierarchy。
RedisConnection通过RedisConnectionFactory来生成，这个工厂方法有类似于PersistenceExceptionTranslator的作用
